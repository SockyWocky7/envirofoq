# Envirofoq

## Envirofoq is a fog mod built around compatibility with voxy Backports, but also works standalone.
This mod is NOT proven to be compatible with sodium version 8. this has been tested for sodium 6.13.
Support for sodium 8 will be added soon.

## Works with voxy ##
*Render distance of 10, voxy RD of 110, fog start 35%, fog end 100%.*
